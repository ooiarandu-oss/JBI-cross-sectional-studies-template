A python-based template to support the methodological quality appraisal of analytical cross-sectional studies using the Joanna Briggs Institute (JBI) Critical Appraisal Checklist.
This project reproduces the original JBI appraisal template while adding digital workflow features for research and systematic review purposes. This project is an independent academic and research-oriented implementation inspired by the original JBI appraisal template.
The Joanna Briggs Institute retains the rights to the original checklist and methodological materials.
